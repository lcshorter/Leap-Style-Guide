Dropdowns

Description

Dropdowns describe the pattern of a toggleable menu that allows users to choose one value from a predefined set.

Usage

Dropdowns are used

When to Use

  • This pattern should be used for selecting multiple options in a set (checkbox)
  • This pattern should be used for selecting one option out of a set (radio button and dropdown)
  • This pattern should be used for toggling an option on and off (switch) ## When Not to Use
  • This pattern does not cover user text input

Detailed Interaction Description

Dropdown Button

A dropdown button is used to make both a single selection as well as take an action. It should be used when there is a subsequent action that is required after making a selection.

Dropdown

Dropdowns are used when a user makes a single selection out of a set. Dropdowns are more space efficient than radio buttons, and should be used when the set is too large to be displayed at once or doesnâ€™t need to be visible at a glance.
