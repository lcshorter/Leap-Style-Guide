The 8px unit was selected based on the way that pixels are treated across screen resolutions. they elements are generally scaled up and down (from 1x, to 2x, to 3x, etc.) and fit to increments of 4 and 8. In addition it creates a maintainable vertical rhythm throughout the interface, because default sizing of web typography is based on a 16px base.

Usage

The spacing increments outlined above are to be used to define the spatial relationships between elements to create and maintain consistency and rhythm.

When to Use

  • This pattern should be used for padding and margin declarations.
  • This pattern should be applied when element has a declared position that is intended to be a set relationship from another element.
